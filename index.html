Slidey

×

You can click and drag blocks to move them left and right.
Blocks can only move into free space (grey).
Blocks fall when there is free space below.
When a row is full the row deletes.
Each time a move is done the blocks all move up.
If the blocks reach the top the game is over.
You can't move blocks in the bottom row, but use it to know where blocks might fall next.
This game goes on forever if you're good enough!
Compare high scores with your friends.
